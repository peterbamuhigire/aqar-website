# Style Brief

## Design Direction
Product-Led Beauty — the website itself proves the software is beautiful and easy to use. Dashboard screenshots are the hero content, presented in elegant device mockups with premium styling.

## Personality
Modern, polished, trustworthy. Technically competent but personally warm. African identity woven into colour and tone without cliche.

## Colour Palette — "Kampala Dusk"

| Role | Hex | Usage |
|------|-----|-------|
| Primary | #0D7377 | Deep Teal — headers, primary CTAs, nav |
| Secondary | #E8913A | Warm Amber — accent CTAs, highlights, badges |
| Dark | #1A1F2E | Charcoal — hero bg, footer, dark sections |
| Light | #FAFAF7 | Warm White — page bg, cards |
| Success | #34A853 | Soft Green — status indicators |
| Text Light | #2D2D2D | Near Black on light backgrounds |
| Text Dark | #F5F5F0 | Off White on dark backgrounds |

## Typography
- **Headings:** Plus Jakarta Sans (700, 800)
- **Body:** Source Sans 3 (400, 600)
- **Monospace:** JetBrains Mono (400) — for any technical UI labels
- All self-hosted via Fontsource

## Visual Language
- Dashboard screenshots in floating device mockups (laptop, tablet, phone) with soft drop shadows and 2-4 degree tilt
- Glass morphism cards (backdrop-blur, subtle white borders) for feature highlights
- Alternating dark charcoal / warm white sections
- Subtle GSAP scroll reveals — elements fade up, screenshots slide in from sides
- No stock illustrations, no abstract blobs. Real screenshots + real photography only.

## Layout
- Dark hero sections make screenshots pop
- Alternating section backgrounds prevent false bottoms
- Feature sections alternate screenshot left/right
- Mobile-first: 375px base, enhanced at 768px and 1280px+

## CTAs
- **Primary:** "Try Demo" — deep teal button, links to property.chwezi.co.za
- **Secondary:** "WhatsApp Us" — warm amber button, links to wa.me/256784464178
- Both CTAs appear in nav, hero, and CTA banner sections

## Sector
Hybrid corporate (structured, feature grids, demo CTAs) + professional services (Peter's story, relationship-driven tone, East African warmth)
