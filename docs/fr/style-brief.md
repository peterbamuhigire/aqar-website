# Brief Stylistique

## Direction de Design
Product-Led Beauty — le site web lui-même prouve que le logiciel est beau et facile à utiliser. Les captures d'écran du tableau de bord sont le contenu vedette, présentées dans d'élégantes maquettes d'appareils avec un style premium.

## Personnalité
Moderne, soigné, digne de confiance. Techniquement compétent mais personnellement chaleureux. Identité africaine tissée dans les couleurs et le ton sans cliché.
